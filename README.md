Simple yarn project to showcase [this issue](https://github.com/bluesky-social/bsky-docs/issues/215)

# setup
if you have [yarn 4](https://yarnpkg.com/cli) installed, you just need to run
```
yarn install
```

# running
```
yarn test
```
